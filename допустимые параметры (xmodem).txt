порядок параметров не имеет значения

ИМЯ_ФАЙЛА          в одиночных кавычках, экранированных (обязательно) двойных, или совсем без них.
				   (какие-нибудь кавычки обязательны только в том случае, если расширение файла отсутствует)
СКОРОСТЬ_ПЕРЕДАЧИ  просто число. По умолчанию = 115200.
РАЗМЕР_БАЙТА       просто число. По умолчанию = 8. (Допустимы значения от 4 до 8)
ИМЯ_ПОРТА          com%НОМЕР_ПОРТА%. По умолчанию = COM0.
КОНТРОЛЬ_ЧЁТНОСТИ  {no, odd, even, mark, space}. По умолчанию отсутствует.
СТОПОВЫЙ_БИТ       {one, half, two}. По умолчанию один. "half" - это полтора, если что. От "half as"
ТЕРМИНАЛЬНЫЙ РЕЖИМ {terminal} - включает терминальный режим. После завершения передачи файла программа продолжит работать в режиме обыкновенного терминала.

для посылки каких-либо команд после инициализации подключения:
    прописывать эти команды по байтам в шестнадцатеричном формате вида 0x## через пробел.
    последовательности команд отделять от следующей любыми другими символами. (самую последнюю тоже)


Пример команды для .bat файла
    xmodem.exe test_file.bin com5 8 921600 two 0xCC 0x00 0x99 0x99 0x01 0x32 , 0xCC 0x00 0xFA 0x00 0xFA ;
